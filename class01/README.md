## Process

同時に動いているアプリ１つ１つをプロセスという

リソースの少ないときからプロセスの概念が存在。
１つのプログラムに１つのマシンではもったいない。
リソースを有効に活用できるからプロセスの概念が使われている。
管理単位としてのプロセス。

複数プロセスの同時実行が難しく努力と労力がかかっている。
プロセス作成からカーネルがからんでくる。
